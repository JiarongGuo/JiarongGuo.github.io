题目描述

一组数，分别表示地平线的高度变化。高度值为整数，相邻高度用直线连接。找出并统计有多少个可能积水的低洼地？

如图：地高变化为 [0,1,0,2,1,2,0,0,2,0][0,1,0,2,1,2,0,0,2,0][0,1,0,2,1,2,0,0,2,0]。

输入格式

两行，第一行 n,n,n, 表示有 nnn 个数。第 222 行连续 nnn 个数表示地平线高度变化的数据，保证首尾为 000。(3≤n≤10000,0≤(3 \le n \le 10000,0 \le (3≤n≤10000,0≤ 高度 ≤1000) \le 1000)≤1000)。

输出格式

一个数，可能积水低洼地的数目。

输入输出样例

输入 #1 
10
0 1 0 2 1 2 0 0 2 0
输出 #1 
3
